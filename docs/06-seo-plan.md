# SEO starter plan

For each client project:

- Create one clear homepage title and description.
- Create service, location, menu, product, ministry, event, or contact pages only when they reflect real approved offerings or locations.
- Use the `SEO.astro` component for metadata and canonical URLs.
- Use `LocalBusinessSchema.astro` as the baseline schema, then adjust `@type` and fields to match the client.
- Confirm Google Business Profile consistency for name, address, phone, hours, and service area.
- Customize `robots.txt` and sitemap URLs before launch.
- Preserve legitimate redirects from the previous site.
