# Integration plan

This starter includes placeholders for common external systems. In real client projects, preserve existing providers unless there is a clear approved reason to replace them.

## Placeholders included

- Church giving
- Church livestream
- Church events
- Restaurant or hospitality ordering
- Appointment booking
- Ecommerce or shop link

## Default rule

Use external systems for payments, ordering, booking, registrations, donations, livestream hosting, inventory, taxes, shipping, refunds, customer accounts, donor records, member records, and staff scheduling.
