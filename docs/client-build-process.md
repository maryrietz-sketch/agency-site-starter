# Client Build Process

1. Audit the current public site and business profiles.
2. Inventory content, assets, integrations, SEO, redirects, and risks.
3. Confirm ownership/permission for assets and claims.
4. Preserve existing operational platforms unless replacement is approved.
5. Build pages, CMS fields, metadata, schema, and integration placeholders.
6. Run checks and document results.
7. Launch only after client approval.
