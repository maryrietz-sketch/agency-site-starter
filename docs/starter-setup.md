# Starter Setup

1. Create or clone a client repo from this starter.
2. Copy `.env.example` to `.env` locally and fill only approved local values.
3. Create a Sanity project and dataset.
4. Configure Cloudflare Pages with build command `npm run build:web`, output `apps/web/dist`, root `/`, branch `main`.
5. Keep mockups private and noindexed until client approval.
