# Sanity CMS model

The starter Studio includes these document types:

- businessInfo
- siteSettings
- page
- service
- menuSection
- menuItem
- productCategory
- product
- testimonial
- galleryImage
- teamMember
- faq
- location
- event

Each client project should remove irrelevant types or hide them from the Studio desk structure if they create editor confusion.

Use approval status fields for reused content, testimonials, gallery images, services, pages, products, events, and team information.
